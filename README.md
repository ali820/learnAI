## 📝 笔记原则

- **Feynman 法** — 用自己的话解释每个概念
- **代码优先** — 每篇笔记都有可运行的 Python 代码
- **链接驱动** — 每篇笔记至少链接 3 篇其他笔记
- **实践闭环** — 每个理论对应一个可做的项目
