# P5-房价预测

> 回归任务的经典之战。从 EDA 到 Stacking，全面提升房价预测精度。

---

## 🎯 项目目标

1. 掌握回归问题的完整 pipeline
2. 学会处理高维混合特征（数值 + 类别 + 缺失）
3. 理解对数变换 y 对回归的影响
4. 尝试多模型集成

---

## 📦 数据

[Kaggle House Prices](https://www.kaggle.com/c/house-prices-advanced-regression-techniques/data)

79 个特征，1460 行训练数据。特点是**混合类型 + 大量缺失**。

---

## 📋 任务步骤

### Step 1：EDA 与数据清洗
- [ ] 分析 SalePrice 的分布（是否正态？需要 log 变换？）
- [ ] 统计缺失值，制定处理策略
- [ ] 找出与房价最相关的 Top 10 特征

### Step 2：特征工程
```python
- [ ] 数值特征：填补缺失（中位数/0）、标准化
- [ ] 类别特征：合并稀有类别、One-Hot 编码
- [ ] 目标编码（可选）：对高基数类别特征
- [ ] 新特征：总面积、房间数/面积比等
```

### Step 3：模型训练

### Step 4：集成与调优

- [ ] 尝试 Stacking：多个基础模型 + 一个 Meta 模型
- [ ] 生成 `submission.csv` 提交 Kaggle
