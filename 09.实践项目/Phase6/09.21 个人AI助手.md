# 终极项目：个人 AI 助手

> 整合你学到的所有技能，打造一个属于你的 AI 助手。

---

## 🎯 项目目标

搭建一个功能完整的个人 AI 助手，具备：
- 🧠 **RAG 记忆**：能记住你的笔记、日程、偏好
- 🔧 **工具使用**：能搜索、计算、查天气、查日历
- 👁️ **多模态**：能看图、识图
- 📱 **多渠道**：Web UI + API

---

## 🏗️ 技术栈

```
前端: Streamlit / Gradio / Next.js
后端: FastAPI + LangChain
模型: Qwen/LLaMA (本地) 或 GPT-4/Claude (API)
向量库: Chroma / Qdrant
部署: Docker + Nginx
```

---

## 🏆 验收标准

- [ ] 能用自然语言完成任务（而非点按钮）
- [ ] 有上下文记忆（多轮对话不丢失信息）
- [ ] 至少 3 个可用工具
- [ ] 部署到公网可访问
