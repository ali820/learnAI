### ⏰ 11:00 - 12:00 | NumPy 初体验 🔥

**今天的重头戏**！NumPy 是 AI 的"算盘"，一切深度学习框架都建立在它的思想上。

```python
import numpy as np

# === 基础操作（今天全部掌握） ===

# 1. 创建数组（4 种方式）
a = np.array([1, 2, 3])           # 从列表
b = np.zeros((3, 4))               # 全零
c = np.ones((2, 3))                # 全一
d = np.arange(10)                  # 序列 [0,1,...,9]
e = np.random.randn(3, 3)          # 标准正态分布随机数

# 2. 数组属性
print(a.shape)   # (3,) — 形状
print(b.ndim)    # 2 — 维度数
print(b.dtype)   # float64 — 数据类型

# 3. 索引和切片
mat = np.arange(12).reshape(3, 4)  # 3x4 矩阵
print(mat[0, :])    # 第一行 → [0, 1, 2, 3]
print(mat[:, 1])    # 第二列 → [1, 5, 9]
print(mat[1:3, :])  # 第2-3行

# 4. 矩阵运算（理解这个，就理解了神经网络的前向传播！）
A = np.random.randn(3, 4)
B = np.random.randn(4, 2)
C = A @ B          # 矩阵乘法！shape: (3, 4) @ (4, 2) = (3, 2)
print(C.shape)

# 5. 广播机制（NumPy 的核心魔法）
data = np.random.randn(5, 3)
mean = data.mean(axis=0)     # shape (3,)
std = data.std(axis=0)        # shape (3,)
normalized = (data - mean) / std  # 广播！(5,3) 自动匹配 (3,)
```

> 💡 **关键直觉**：矩阵乘法 `A @ B` 是神经网络中信息传递的核心操作。输入 X 乘以权重 W → 下一层。今天理解这个，Phase 3 你就比别人快 50%。

### ⏰ 14:00 - 15:00 | 动手：完成第一个完整脚本

```python
# day1_warmup.py
import numpy as np

# 任务：创建一个 5x5 的随机矩阵，完成以下操作
np.random.seed(42)  # 固定随机种子，确保可复现
mat = np.random.randn(5, 5)

print("原始矩阵：")
print(mat)

print("\n1. 每行的和：")
print(mat.sum(axis=1))   # axis=1 表示沿列方向求和 = 每行

print("\n2. 每列的平均值：")
print(mat.mean(axis=0))  # axis=0 表示沿行方向求平均 = 每列

print("\n3. 矩阵的转置：")
print(mat.T)

print("\n4. 矩阵乘以转置（得到一个对称矩阵）：")
print(mat @ mat.T)  # (5,5) @ (5,5) = (5,5) 对称矩阵！
```
