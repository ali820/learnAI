# P20-论文复现

> 从"读论文"到"复现论文"。这是通往独立研究的必经之路。

---

## 🎯 项目目标

1. 选择一篇顶会论文并完整复现
2. 读懂论文的每一个公式和实现细节
3. 在自己的数据上跑出可比的实验结果
4. 尝试一个小的改进

---

## 📄 推荐论文（由易到难）

| 论文 | 领域 | 难度 |
|------|------|------|
| ResNet (2015) | CV | ⭐⭐ |
| Transformer (2017) | NLP | ⭐⭐⭐ |
| CLIP (2021) | 多模态 | ⭐⭐⭐ |
| LoRA (2021) | LLM | ⭐⭐⭐ |
| LLaMA (2023) | LLM | ⭐⭐⭐⭐ |

---

## 📋 任务步骤

### Step 1：论文精读
- [ ] 用 [[10.论文阅读/10.01 论文笔记模板]] 写论文笔记
- [ ] 画出算法流程图
- [ ] 列出所有你不理解的细节 → 逐一查清楚

### Step 2：复现
- [ ] 找官方代码或开源实现作为参考
- [ ] 从零写核心算法（不看参考代码）
- [ ] 在相同数据集上跑出可比结果

### Step 3：改进
- [ ] 思考：论文的局限性是什么？
- [ ] 尝试一个小改进（改 loss / 改架构 / 改数据）
- [ ] 对比改进前后的效果

---

## 🏆 验收标准

- [ ] 核心算法能跑通并产生合理结果
- [ ] 与论文报告的指标差距 < 5%
- [ ] 完成论文笔记（含批判性思考）
