# P9-情感分析

> 用 LSTM 读懂电影评论。这是你第一次用深度学习处理文本。

---

## 🎯 项目目标

1. 掌握文本分类的 DL pipeline
2. 对比 LSTM vs GRU vs 双向 LSTM
3. 理解词嵌入的可视化
4. 准确率 > 85%

---

## 📦 数据

[IMDB 数据集](https://huggingface.co/datasets/imdb) — 25000 训练 + 25000 测试，二分类（正面/负面）。

---

## 📋 任务步骤

### Step 1：数据预处理
- [ ] 分词 + 构建词汇表
- [ ] 填充/截断到固定长度
- [ ] 分析：评论长度分布？最常见的词是什么？

### Step 2：模型对比
```python
# 训练 3 个模型，记录准确率和训练时间：
models = {
    'LSTM': ...,
    'GRU': ...,
    'BiLSTM': ...,   # 双向 LSTM
}
```

### Step 3：词嵌入可视化
- [ ] 取训练好的 Embedding 层
- [ ] 用 PCA/t-SNE 降到 2D
- [ ] 标出情感相关的词（"amazing", "terrible", "boring" 等）

### Step 4：错误分析
- [ ] 找出模型最自信但预测错的 10 个样本
- [ ] 分析：是标注问题？还是反讽/双重否定？
