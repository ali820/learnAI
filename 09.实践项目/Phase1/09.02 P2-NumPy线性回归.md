# P2-NumPy 线性回归

> 不用 Sklearn，只用 NumPy 从零实现线性回归。这是理解一切 ML 模型训练的起点。

---

## 🎯 项目目标

1. 用**闭式解**（正规方程）实现线性回归
2. 用**梯度下降**实现线性回归
3. 对比两种方法的结果和优缺点
4. 可视化数据 + 拟合线 + 损失下降曲线

---

## 📦 数据

```python
import numpy as np
import matplotlib.pyplot as plt

# 生成模拟数据：y = 3x₁ + 1.5x₂ - 2 + 噪声
np.random.seed(42)
n = 200
X = np.random.randn(n, 2)
true_w = np.array([3.0, 1.5])
true_b = -2.0
y = X @ true_w + true_b + np.random.randn(n) * 0.5
```

---

## 📋 任务步骤

### Step 1：闭式解（正规方程）

$$\hat{w} = (X^T X)^{-1} X^T y$$

```python
# 添加偏置列
X_b = np.c_[np.ones((n, 1)), X]  # shape: (200, 3)

# 正规方程求解
w_closed = np.linalg.inv(X_b.T @ X_b) @ X_b.T @ y

b_closed = w_closed[0]
w_closed = w_closed[1:]
print(f"闭式解: w = {w_closed}, b = {b_closed:.3f}")
print(f"真值:   w = {true_w}, b = {true_b}")
```

### Step 2：梯度下降求解

```python
def mse_loss(X, y, w, b):
    pred = X @ w + b
    return np.mean((pred - y) ** 2)

def mse_grad(X, y, w, b):
    N = len(X)
    pred = X @ w + b
    error = pred - y
    dw = (2 / N) * (X.T @ error)
    db = (2 / N) * np.sum(error)
    return dw, db

# 训练
w_gd = np.random.randn(2) * 0.01
b_gd = 0.0
lr = 0.05
history = []

for i in range(500):
    loss = mse_loss(X, y, w_gd, b_gd)
    dw, db = mse_grad(X, y, w_gd, b_gd)
    w_gd -= lr * dw
    b_gd -= lr * db
    history.append(loss)

print(f"梯度下降: w = {w_gd}, b = {b_gd:.3f}")
```

### Step 3：可视化对比

```python
fig, axes = plt.subplots(1, 2, figsize=(14, 5))

# 损失下降曲线
axes[0].plot(history)
axes[0].set_xlabel('Iteration')
axes[0].set_ylabel('MSE Loss')
axes[0].set_title('Gradient Descent Convergence')
axes[0].grid(True, alpha=0.3)

# 预测 vs 真实
axes[1].scatter(y, X @ w_gd + b_gd, alpha=0.5, s=10)
axes[1].plot([y.min(), y.max()], [y.min(), y.max()], 'r--', lw=2)
axes[1].set_xlabel('True y')
axes[1].set_ylabel('Predicted y')
axes[1].set_title('Predictions vs True Values')
axes[1].grid(True, alpha=0.3)

plt.tight_layout()
plt.show()
```
