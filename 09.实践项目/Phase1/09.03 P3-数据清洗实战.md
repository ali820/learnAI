# P3-数据清洗实战

> 真实世界的数据从来不是干净的。这个项目模拟一次"脏数据"清洗全流程。

---

## 🎯 项目目标

1. 识别并处理各类常见数据问题
2. 掌握数据清洗的标准流程
3. 学会写数据清洗报告

---

## 📦 数据生成（模拟脏数据）

```python
import pandas as pd
import numpy as np

# 模拟一个电商订单数据集
np.random.seed(123)
n = 1000

df = pd.DataFrame({
    'order_id': range(1, n+1),
    'customer_name': np.random.choice(['张三', '李四', '王五', 'zhao liu', None], n),
    'amount': np.random.choice([100, 200, 350, 500, -50, 99999, None], n),
    'status': np.random.choice(['已完成', 'pending', 'Canceled', '已取消', None], n),
    'order_date': np.random.choice(['2024-01-15', '2024/02/30', '20240301', 'invalid', None], n),
    'phone': np.random.choice(['13800138000', '1380013800', '138001380000', 'abc-defg', None], n),
})

print("=== 原始数据（前10行）===")
print(df.head(10))
print(f"\n数据形状: {df.shape}")
print(f"各列缺失值:\n{df.isnull().sum()}")
```

---

## 📋 任务步骤

### Step 1：问题诊断

先回答以下问题：

```python
# 1. 检查每一列的数据类型
print(df.dtypes)

# 2. 检查每一列的缺失值数量
print(df.isnull().sum())

# 3. 检查每一列的异常值
print(df['amount'].describe())  # 有负数？有99999？

# 4. 检查字符串列的不一致
print(df['status'].unique())
print(df['order_date'].unique())

# 5. 检查 phone 列的格式
print(df['phone'].str.len().value_counts())
```

### Step 2：清洗每一列

```python
# === amount: 移除异常值 ===
# 去掉空值
df = df.dropna(subset=['amount'])
# 金额应该 > 0 且 < 10000
df = df[(df['amount'] > 0) & (df['amount'] < 10000)]

# === status: 统一格式 ===
status_map = {
    '已完成': 'completed',
    'Canceled': 'cancelled',
    '已取消': 'cancelled',
    'pending': 'pending',
}
df['status'] = df['status'].map(status_map).fillna('unknown')

# === order_date: 统一日期格式 ===
df['order_date'] = pd.to_datetime(df['order_date'], errors='coerce')

# === phone: 统一手机号 ===
df['phone'] = df['phone'].astype(str).str.extract(r'(\d{11})')[0]

# === customer_name: 处理缺失 ===
df['customer_name'] = df['customer_name'].fillna('未知用户')
```
