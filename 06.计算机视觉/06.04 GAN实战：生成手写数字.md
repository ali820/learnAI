# GAN 实战：生成手写数字

> 生成对抗网络 = 一个造假者 + 一个鉴定师。二者在博弈中共同进化。

---

## 🎯 练习目标

1. 理解 GAN 的博弈论思想（生成器 vs 判别器）
2. 从零实现 DCGAN
3. 训练生成 MNIST 手写数字
4. 理解 GAN 训练的难点（模式坍塌、训练不稳定）

---

## 1️⃣ GAN 的核心思想

```
Generator（造假者）:           Discriminator（鉴定师）:
随机噪声 z → 假图片            图片 → 真 or 假？

训练过程:
1. 给 Discriminator 看真图 → 输出 1（真）
2. 给 Discriminator 看 G(z) → 输出 0（假）
3. Discriminator 变强 → 更难骗
4. Generator 变强 → 更逼真
5. 循环 → 直到 G 生成的图 Discriminator 分不出真假
```

```python
# GAN 的数学本质（最小最大博弈）:
# min_G max_D  E[log D(x)] + E[log(1 - D(G(z)))]
#  ↑        ↑
# G想让D犯错   D想最大化区分能力
```

## 2️⃣ 🔥 DCGAN 完整实现

```python
import torch
import torch.nn as nn
import torch.optim as optim
from torchvision import datasets, transforms
from torch.utils.data import DataLoader
import matplotlib.pyplot as plt

# === Generator ===
class Generator(nn.Module):
    """噪声 → 图片（转置卷积上采样）"""
    def __init__(self, latent_dim=100):
        super().__init__()
        self.main = nn.Sequential(
            # 100 → 128×7×7
            nn.ConvTranspose2d(latent_dim, 128, 7, 1, 0, bias=False),
            nn.BatchNorm2d(128),
            nn.ReLU(True),
            # → 64×14×14
            nn.ConvTranspose2d(128, 64, 4, 2, 1, bias=False),
            nn.BatchNorm2d(64),
            nn.ReLU(True),
            # → 1×28×28
            nn.ConvTranspose2d(64, 1, 4, 2, 1, bias=False),
            nn.Tanh()  # 输出归一化到 [-1, 1]
        )
    
    def forward(self, z):
        return self.main(z)

# === Discriminator ===
class Discriminator(nn.Module):
    """图片 → 真/假分数（卷积下采样）"""
    def __init__(self):
        super().__init__()
        self.main = nn.Sequential(
            # 1×28×28 → 64×14×14
            nn.Conv2d(1, 64, 4, 2, 1, bias=False),
            nn.LeakyReLU(0.2, inplace=True),
            # → 128×7×7
            nn.Conv2d(64, 128, 4, 2, 1, bias=False),
            nn.BatchNorm2d(128),
            nn.LeakyReLU(0.2, inplace=True),
            # → 1
            nn.Conv2d(128, 1, 4, 1, 0, bias=False),
            nn.Sigmoid()
        )
    
    def forward(self, x):
        return self.main(x).view(-1, 1)

# === 初始化 ===
device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
G = Generator().to(device)
D = Discriminator().to(device)

criterion = nn.BCELoss()
optimizer_G = optim.Adam(G.parameters(), lr=0.0002, betas=(0.5, 0.999))
optimizer_D = optim.Adam(D.parameters(), lr=0.0002, betas=(0.5, 0.999))

# 加载 MNIST
transform = transforms.Compose([
    transforms.ToTensor(),
    transforms.Normalize([0.5], [0.5])  # → [-1, 1]
])
dataloader = DataLoader(
    datasets.MNIST('./data', train=True, download=True, transform=transform),
    batch_size=128, shuffle=True
)
```

## 3️⃣ GAN 训练循环（和之前所有训练都不一样！）

```python
fixed_noise = torch.randn(64, 100, 1, 1, device=device)

for epoch in range(50):
    for i, (real_imgs, _) in enumerate(dataloader):
        real_imgs = real_imgs.to(device)
        batch_size = real_imgs.size(0)
        
        # === 1. 训练 Discriminator ===
        # 真图 → label=1
        real_labels = torch.ones(batch_size, 1, device=device)
        output_real = D(real_imgs)
        loss_D_real = criterion(output_real, real_labels)
        
        # 假图 → label=0
        noise = torch.randn(batch_size, 100, 1, 1, device=device)
        fake_imgs = G(noise)
        fake_labels = torch.zeros(batch_size, 1, device=device)
        output_fake = D(fake_imgs.detach())  # detach! 不更新 G
        loss_D_fake = criterion(output_fake, fake_labels)
        
        loss_D = loss_D_real + loss_D_fake
        optimizer_D.zero_grad()
        loss_D.backward()
        optimizer_D.step()
        
        # === 2. 训练 Generator ===
        # 假图 → 骗过 D → label=1
        noise = torch.randn(batch_size, 100, 1, 1, device=device)
        fake_imgs = G(noise)
        output = D(fake_imgs)
        loss_G = criterion(output, real_labels)  # G 希望 D 判断为真
        
        optimizer_G.zero_grad()
        loss_G.backward()
        optimizer_G.step()
    
    if epoch % 10 == 0:
        print(f"Epoch {epoch:2d} | D loss: {loss_D:.4f} | G loss: {loss_G:.4f}")
```

## 5️⃣ GAN 训练的常见坑

| 问题 | 症状 | 解决方案 |
|------|------|---------|
| 模式坍塌 | G 只会生成几种样本 | WGAN-GP、增加多样性损失 |
| D 太强 | G 的 loss 不降 | 降低 D 的学习率 / 每轮多训几次 G |
| G 太强 | D 的准确率接近 50% | 正常！说明博弈进入均衡 |
| 训练震荡 | loss 上蹿下跳 | 降低学习率、用 spectral norm |

```python
# WGAN 改进版损失（解决模式坍塌）
# D 的损失: -E[D(real)] + E[D(fake)]        （不用 sigmoid）
# G 的损失: -E[D(G(z))]
# 再加 Gradient Penalty → WGAN-GP

print("💡 DCGAN → WGAN-GP → StyleGAN → Stable Diffusion")
print("   GAN 的核心思想没变，变的是稳定训练的技巧")
```
