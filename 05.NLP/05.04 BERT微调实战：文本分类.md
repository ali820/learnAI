## 1️⃣ BERT 是怎么训练的？

```
BERT = Transformer Encoder × 12（或 24）层

预训练任务 1 — Masked Language Model (MLM):
  输入: "我 [MASK] 吃苹果"
  预测: "爱"  （15% 的词被随机遮盖，让模型猜）

预训练任务 2 — Next Sentence Prediction (NSP):
  输入: 句子A + [SEP] + 句子B
  预测: B 是不是 A 的下一句？
```


## 2️⃣ 🔥 微调 BERT 做文本分类（完整流程）

```python
import torch
import torch.nn as nn
from transformers import AutoModel, AutoTokenizer
from torch.utils.data import DataLoader, Dataset
from datasets import load_dataset

# === 1. 加载数据 ===
dataset = load_dataset('imdb')
print(f"训练集: {len(dataset['train'])} 条")

# === 2. 定义 Dataset ===
class IMDBDataset(Dataset):
    def __init__(self, texts, labels, tokenizer, max_len=256):
        self.texts = texts
        self.labels = labels
        self.tokenizer = tokenizer
        self.max_len = max_len
    
    def __len__(self):
        return len(self.texts)
    
    def __getitem__(self, idx):
        text = str(self.texts[idx])
        label = self.labels[idx]
        
        encoding = self.tokenizer(
            text,
            max_length=self.max_len,
            padding='max_length',
            truncation=True,
            return_tensors='pt'
        )
        
        return {
            'input_ids': encoding['input_ids'].squeeze(0),
            'attention_mask': encoding['attention_mask'].squeeze(0),
            'label': torch.tensor(label, dtype=torch.long)
        }

# === 3. 定义分类模型 ===
class BertClassifier(nn.Module):
    def __init__(self, model_name='bert-base-uncased', num_classes=2, dropout=0.3):
        super().__init__()
        self.bert = AutoModel.from_pretrained(model_name)
        self.dropout = nn.Dropout(dropout)
        self.classifier = nn.Linear(self.bert.config.hidden_size, num_classes)
        # 冻结 BERT 底层（可选）
        # for param in self.bert.parameters():
        #     param.requires_grad = False
    
    def forward(self, input_ids, attention_mask):
        outputs = self.bert(input_ids=input_ids, attention_mask=attention_mask)
        pooled = outputs.pooler_output  # [CLS] token 的输出（代表整句）
        pooled = self.dropout(pooled)
        return self.classifier(pooled)

# === 4. 训练（复用 04.03 的标准模板） ===
print("💡 训练代码和 04.03 PyTorch 入门实战完全一样！")
print("   只换了三个东西：模型、数据、Tokenizer")
print("   train_epoch() / evaluate() 模板照搬即可。")
```

## 4️⃣ HuggingFace Pipeline：一行代码搞定

```python
from transformers import pipeline

# 文本分类 — 零代码！
classifier = pipeline('sentiment-analysis')
print(classifier("This movie is fantastic!"))
print(classifier("I really hate this product."))

# 命名实体识别
ner = pipeline('ner', grouped_entities=True)
print(ner("Apple was founded by Steve Jobs in California."))

# 问答
qa = pipeline('question-answering')
context = "The Eiffel Tower is located in Paris, France. It was built in 1889."
print(qa(question="When was the Eiffel Tower built?", context=context))
```

## 5️⃣ 微调技巧总结

| 技巧 | 说明 |
|------|------|
| 学习率要小 | BERT 微调用 2e-5 ~ 5e-5，太大破坏预训练知识 |
| Warmup | 前 10% steps 线性增加学习率 |
| 层冻结 | 先冻结底层，只训分类头 2 epoch，再解冻全模型 |
| 长文本 | 超过 512 token 时：截断 / 滑动窗口 / 用 Longformer |
| Batch Size | 尽量大（但受显存限制），梯度累积模拟大 batch |
