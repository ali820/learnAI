# 🛠️ Python 环境搭建实战

> 手把手搭建 AI 学习所需的完整 Python 环境。

---

## 为什么不用系统自带 Python？

AI 项目的依赖极其复杂。不同项目可能需要不同版本的 NumPy、PyTorch。**虚拟环境**能让你每个项目有独立的包集合，互不干扰。

---

## 📦 方案选择

| 方案 | 适用场景 | 推荐度 |
|------|---------|--------|
| **Miniconda** + `pip` | 通用方案，兼顾包管理和虚拟环境 | ⭐⭐⭐⭐⭐ |
| Anaconda（完整版） | 需要预装 1500+ 包的新手 | ⭐⭐⭐⭐ |
| 系统 Python + venv | 轻量需求，但需手动处理 C 扩展 | ⭐⭐⭐ |

> 🏆 推荐：**Miniconda** — 只装核心，需要什么再用 `pip` 装什么。

---

## 🚀 Step-by-Step 安装

### Step 1：安装 Miniconda

**Windows：**
1. 访问 https://docs.conda.io/en/latest/miniconda.html
2. 下载 `Miniconda3 Windows 64-bit`
3. 双击安装，**全部默认选项**
4. 安装完成后，搜索打开 **Anaconda Prompt**

**Mac/Linux：**
```bash
# Mac
curl -O https://repo.anaconda.com/miniconda/Miniconda3-latest-MacOSX-arm64.sh
bash Miniconda3-latest-MacOSX-arm64.sh

# Linux
curl -O https://repo.anaconda.com/miniconda/Miniconda3-latest-Linux-x86_64.sh
bash Miniconda3-latest-Linux-x86_64.sh
```

验证安装：
```bash
conda --version
# 输出：conda 24.x.x
```

### Step 2：创建 AI 学习专用环境

```bash
# 创建环境（Python 3.11 是当前稳定版本）
conda create -n ai-learning python=3.11

# 激活环境
conda activate ai-learning

# 你的终端前面应该出现 (ai-learning) 前缀
```

### Step 3：安装核心库

```bash
# Phase 1-2 必备
pip install numpy pandas matplotlib seaborn scipy jupyter

# 验证安装
python -c "import numpy; import pandas; import matplotlib; print('All OK!')"
```

### Step 4：安装 PyTorch（Phase 3 会用，先装好）

```bash
# CPU 版本（没有 NVIDIA 显卡）
pip install torch torchvision torchaudio

# GPU 版本（有 NVIDIA 显卡，需先装 CUDA）
# 前往 https://pytorch.org/get-started/locally/ 获取适合你系统的命令
```

**检查 GPU 是否可用：**
```bash
python -c "import torch; print('GPU Available:', torch.cuda.is_available())"
```

### Step 5：配置 Jupyter

```bash
# 生成配置文件
jupyter notebook --generate-config

# 启动 Jupyter
jupyter notebook

# 或启动 Jupyter Lab（更现代）
jupyter lab
```

> 💡 **建议**：把 Jupyter 的工作目录设为你的 vault 根目录，方便在笔记和代码间切换。

### Step 6：安装 VS Code（可选但推荐）

VS Code + Python 插件 + Jupyter 插件 = 最佳 AI 学习 IDE。

1. 下载 VS Code：https://code.visualstudio.com/
2. 安装扩展：
   - **Python** (Microsoft)
   - **Jupyter** (Microsoft)
   - **Pylance** (Microsoft)

---

## 🔧 配置 VS Code 联动 Conda

打开 VS Code → `Ctrl+Shift+P` → 输入 `Python: Select Interpreter` → 选择包含 `ai-learning` 的 Python。

---

## 🔁 日常使用流程

```bash
# 打开终端，激活环境
conda activate ai-learning

# 启动 Jupyter
jupyter lab

# 或启动 VS Code 后直接新建 .ipynb 文件

# 每次开始学习前，更新核心库
pip install --upgrade numpy pandas matplotlib torch
```

---

## ❓ 常见问题

| 问题 | 解决方法 |
|------|---------|
| `conda` 不是内部命令 | 重启终端 / 检查 PATH 环境变量 |
| `pip` 安装很慢 | 使用国内镜像源：`pip install -i https://pypi.tuna.tsinghua.edu.cn/simple xxx` |
| Jupyter 内核找不到 conda 环境 | `ipython kernel install --user --name=ai-learning` |
| `torch.cuda.is_available()` 返回 False | 检查 CUDA 版本是否匹配 PyTorch 要求 |
