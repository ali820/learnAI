# P12-目标检测

> 在自己的数据集上训练一个 YOLO 检测器，感受从标注到部署的全流程。

---

## 🎯 项目目标

1. 准备自定义检测数据集（标注 + 格式转换）
2. 训练 YOLO 模型
3. 验证并分析检测结果
4. 导出模型用于推理

---

## 📦 数据

**方案 A（推荐）**：从 [Roboflow Universe](https://universe.roboflow.com/) 找一个公开检测数据集

**方案 B**：自己标注（用 [LabelImg](https://github.com/HumanSignal/labelImg) 或 [CVAT](https://www.cvat.ai/)）

---

## 📋 任务步骤

### Step 1：数据准备
- [ ] 收集/选择图像（至少 100 张）
- [ ] 标注边界框
- [ ] 转换成 YOLO 格式
- [ ] 划分 train/val/test

### Step 2：训练
```python
from ultralytics import YOLO
model = YOLO('yolo11n.pt')
model.train(data='data.yaml', epochs=50, imgsz=640)
```

### Step 3：评估与分析
- [ ] 查看 mAP、Precision、Recall 曲线
- [ ] 混淆矩阵：模型最常把什么认成什么？
- [ ] 找出 FP（误检）和 FN（漏检）的样本

### Step 4：部署
- [ ] 导出 ONNX 格式
- [ ] 写一个简单的推理脚本（摄像头/视频/图片）

---

## 🏆 验收标准

- [ ] mAP@0.5 > 0.5（自定义数据集）
- [ ] 推理速度 > 10 FPS
- [ ] 能分析检测失败的原因
