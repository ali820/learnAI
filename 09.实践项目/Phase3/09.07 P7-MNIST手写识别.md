# P7-MNIST 手写识别

> 深度学习版的"Hello World"。用 PyTorch 搞定手写数字识别，建立你的第一个 DL pipeline。

---

## 🎯 项目目标

1. 掌握 PyTorch 完整训练流程
2. 对比 NumPy 手写 vs PyTorch 的效率和代码量
3. 学会调参（学习率、batch size、隐藏层大小）
4. 准确率 > 97%

---

## 📦 数据

`torchvision.datasets.MNIST` — 60000 训练 + 10000 测试，28×28 灰度图，10 类。

---

## 📋 任务步骤

### Step 1：NumPy 版 baseline
- [ ] 用 [[04.深度学习/04.01 从零实现多层感知机|04.01]] 的 `TwoLayerNet` 训练
- [ ] 记录：代码行数、训练时间、最终准确率

### Step 2：PyTorch 版
- [ ] 用 [[04.深度学习/04.03 PyTorch入门实战：MNIST分类|04.03]] 的 `MLP` 训练
- [ ] 对比 NumPy 版：代码少了多少？准确率高了多少？

### Step 3：调参实验（至少做 3 组）

| 参数 | 基线 | 实验 1 | 实验 2 |
|------|------|--------|--------|
| hidden_dim | 128 | 256 | 512 |
| learning_rate | 0.01 | 0.001 | 0.1 |
| batch_size | 64 | 32 | 128 |

- [ ] 每组记录 test accuracy
- [ ] 分析：哪个参数对结果影响最大？

### Step 4：可视化
- [ ] 画出训练/测试的 loss 和 accuracy 曲线
- [ ] 显示 10 个预测错误的样本（图 + 预测标签 + 真实标签）
