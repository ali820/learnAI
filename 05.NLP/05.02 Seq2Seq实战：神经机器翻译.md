## 1️⃣ Encoder-Decoder 直觉

```
Encoder（编码器）:              Decoder（解码器）:
"我 爱 机器 学习"               <sos> I love ...
    ↓ 逐词处理                      ↓ 逐词生成
[ 上下文向量 c ]  ─────────→  [ 基于 c 生成下一个词 ]
```

```python
# 核心思想
sentence = "I love machine learning"

# Encoder: 读入整个句子，压缩成一个"思想向量"
context = encode(sentence)  # 浓缩了整句话的含义

# Decoder: 从"思想向量"中解码出目标语言
translation = decode(context)  # "我爱机器学习"
```

## 2️⃣ Attention：让 Decoder 学会"看哪里"

```python
import numpy as np
import matplotlib.pyplot as plt
import seaborn as sns

# 模拟 Attention 权重矩阵
# 行 = 目标语言词（中文），列 = 源语言词（英文）
attention_weights = np.array([
    [0.8, 0.1, 0.1, 0.0],  # "我" 主要关注 "I"
    [0.1, 0.7, 0.1, 0.1],  # "爱" 主要关注 "love"
    [0.0, 0.1, 0.8, 0.1],  # "机器" 关注 "machine"
    [0.0, 0.1, 0.1, 0.8],  # "学习" 关注 "learning"
])

fig, ax = plt.subplots(figsize=(8, 6))
sns.heatmap(attention_weights, annot=True, fmt='.1f', cmap='YlOrRd',
            xticklabels=['I', 'love', 'machine', 'learning'],
            yticklabels=['我', '爱', '机器', '学习'])
ax.set_xlabel('Source (English)')
ax.set_ylabel('Target (Chinese)')
ax.set_title('Attention Weights: Decoder 每一步"看"哪里')
plt.show()

print("💡 Attention 解决了 Seq2Seq 的最大问题：")
print("   固定长度的上下文向量 → 动态的加权上下文向量")
print("   翻译长句时，Decoder 知道每一步该看源句的哪个部分")
```

## 3️⃣ Bahdanau Attention 公式

```
Attention Score:  e_ij = v^T · tanh(W·s_i + U·h_j)
                   ↑         ↑              ↑
               对齐分数    decoder状态    encoder状态

Attention Weight: α_ij = softmax(e_i)  （归一化 → 概率分布）

Context Vector:    c_i = Σ α_ij · h_j  （加权和）
```

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class BahdanauAttention(nn.Module):
    """Additive Attention (Bahdanau 2015)"""
    def __init__(self, hidden_size):
        super().__init__()
        self.W = nn.Linear(hidden_size, hidden_size, bias=False)
        self.U = nn.Linear(hidden_size, hidden_size, bias=False)
        self.v = nn.Linear(hidden_size, 1, bias=False)
    
    def forward(self, decoder_hidden, encoder_outputs):
        # decoder_hidden: [batch, hidden]
        # encoder_outputs: [batch, seq_len, hidden]
        
        seq_len = encoder_outputs.size(1)
        decoder_hidden = decoder_hidden.unsqueeze(1).repeat(1, seq_len, 1)
        
        energy = torch.tanh(self.W(decoder_hidden) + self.U(encoder_outputs))
        attention = self.v(energy).squeeze(-1)    # [batch, seq_len]
        
        return F.softmax(attention, dim=-1)

# 测试
attn = BahdanauAttention(hidden_size=128)
dec_hidden = torch.randn(4, 128)         # batch=4
enc_outputs = torch.randn(4, 10, 128)    # seq_len=10
weights = attn(dec_hidden, enc_outputs)
print(f"Attention weights shape: {weights.shape}")  # [4, 10]
print(f"每行和为 1: {weights[0].sum():.4f}")
```

## 5️⃣ 训练技巧

```python
# Teacher Forcing: 训练时 Decoder 的输入用真实目标词（而不是自己的预测）
# ratio=0.5 → 50% 概率用真实词，50% 用自己的预测
# 优点：加速收敛；缺点：训练和推理时行为不一致 → 可能导致 exposure bias

# Beam Search: 解码时不要 greedy（每次选最大概率），保留 top-k 条路径
# beam_width=3 → 每步保留 3 个候选序列
# 优点：生成质量更高；缺点：计算量大
```
