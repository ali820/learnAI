# LangChain 实战：搭建 AI 助手

> 把 LLM 从"聊天机器人"升级为"能用工具的智能代理"。Agent = LLM + 工具 + 循环决策。

2. 掌握 LangChain 的 Chain / Tool / Agent 核心组件

## 1️⃣ ReAct 模式：Agent 的思考-行动循环

```
用户: "2024 年诺贝尔物理学奖得主多少岁？"

ReAct 循环:
  Thought: 我需要知道 2024 年诺贝尔物理学奖得主是谁
  Action: search("2024 Nobel Prize Physics winner")
  Observation: "John Hopfield, Geoffrey Hinton, 91 岁和 77 岁"
  Thought: 我已经得到了年龄信息
  Final Answer: "2024 年诺贝尔物理学奖得主分别是 John Hopfield (91岁) 
               和 Geoffrey Hinton (77岁)"
```

```python
# pip install langchain langchain-openai
from langchain_openai import ChatOpenAI
from langchain.agents import AgentExecutor, create_react_agent
from langchain.tools import Tool
from langchain.prompts import PromptTemplate
from langchain.memory import ConversationBufferMemory
```

## 2️⃣ 🔥 搭建第一个 Agent（搜索 + 计算）

```python
# === 定义工具 ===

def search_tool(query: str) -> str:
    """模拟搜索工具（实际项目接 Google/Bing API）"""
    # 简化：返回一个硬编码结果
    knowledge_base = {
        "北京天气": "北京今天晴，25°C",
        "python": "Python 3.12 是当前最新版本",
    }
    return knowledge_base.get(query.lower(), f"未找到关于'{query}'的信息")

def calculator(expr: str) -> str:
    """安全的数学计算"""
    try:
        result = eval(expr, {"__builtins__": {}}, {})
        return str(result)
    except Exception as e:
        return f"计算错误: {e}"

tools = [
    Tool(name="Search", func=search_tool, description="搜索信息。输入查询关键词。"),
    Tool(name="Calculator", func=calculator, description="计算数学表达式。输入如 '2+3*4'。"),
]

# === 创建 Agent ===
llm = ChatOpenAI(model="gpt-4o-mini", temperature=0)  # 或任何兼容的 LLM

# ReAct Prompt 模板
react_prompt = PromptTemplate.from_template("""你是一个有用的 AI 助手。你可以使用以下工具：

{tools}

使用以下格式回答：

Question: 用户的问题
Thought: 我需要做什么？
Action: 工具名称
Action Input: 工具输入
Observation: 工具返回结果
... (这个 Thought/Action/Action Input/Observation 可以重复)
Thought: 我现在知道最终答案了
Final Answer: 对用户的最终回答

开始！

Question: {input}
Thought: {agent_scratchpad}""")

agent = create_react_agent(llm, tools, react_prompt)
agent_executor = AgentExecutor(
    agent=agent,
    tools=tools,
    verbose=True,
    handle_parsing_errors=True,
    max_iterations=5,  # 防止无限循环
)

# 测试
response = agent_executor.invoke({"input": "计算 (15 + 7) × 3 的结果"})
print(response["output"])
```

## 3️⃣ Chain：把多个组件串联

```python
from langchain.chains import LLMChain, SequentialChain
from langchain.prompts import ChatPromptTemplate

# Chain 1: 生成大纲
outline_prompt = ChatPromptTemplate.from_template(
    "为主题'{topic}'写一个 3 点大纲"
)
outline_chain = LLMChain(llm=llm, prompt=outline_prompt, output_key="outline")

# Chain 2: 根据大纲写文章
article_prompt = ChatPromptTemplate.from_template(
    "根据以下大纲写一篇 200 字的文章:\n{outline}"
)
article_chain = LLMChain(llm=llm, prompt=article_prompt, output_key="article")

# 串联
pipeline = SequentialChain(
    chains=[outline_chain, article_chain],
    input_variables=["topic"],
    output_variables=["outline", "article"],
)

result = pipeline.invoke({"topic": "人工智能的未来"})
print(result["article"][:300])
```

## 4️⃣ Memory：让 Agent 记住对话

```python
from langchain.memory import ConversationBufferMemory, ConversationSummaryMemory
from langchain.chains import ConversationChain

# 简单记忆：保存完整对话历史
memory = ConversationBufferMemory(return_messages=True)
conversation = ConversationChain(llm=llm, memory=memory)

print(conversation.predict(input="我叫小明"))
print(conversation.predict(input="我刚才说我的名字是什么？"))
# → "你刚才说你叫小明"

# 摘要记忆：长对话时自动压缩历史
summary_memory = ConversationSummaryMemory(llm=llm, return_messages=True)
# 当对话历史超过 token 限制时，自动生成摘要
```
