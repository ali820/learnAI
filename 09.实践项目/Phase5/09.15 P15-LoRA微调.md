# P15-LoRA 微调

> 用一个开源模型 + 你自己的数据，微调出一个专属 LLM。

---

## 🎯 项目目标

1. 用 LoRA 微调一个开源 LLM
2. 让模型学会特定领域的知识或风格
3. 对比微调前后的输出质量

---

## 📦 模型（任选）

| 模型 | 大小 | 显存需求 |
|------|------|---------|
| Qwen2-0.5B | 0.5B | ~2GB (4-bit) |
| Qwen2-1.5B | 1.5B | ~3GB (4-bit) |
| LLaMA3-8B | 8B | ~6GB (4-bit) |

---

## 📋 任务步骤

### Step 1：数据准备
- [ ] 收集/构造 100-500 条训练数据
- [ ] 格式化成 Alpaca/ShareGPT 格式
- [ ] 验证数据质量（有无格式错误、有无低质样本）

### Step 2：LoRA 微调
- [ ] 配置 4-bit 量化 + LoRA（r=16, alpha=32）
- [ ] 训练 3-5 epochs
- [ ] 记录 loss 曲线

### Step 3：对比评估
- [ ] 准备 10 个测试 Prompt
- [ ] 分别用基础模型和微调模型生成
- [ ] 人工打分（1-5 分）或 用 GPT-4 评判

### Step 4：分析
- [ ] 微调有没有导致灾难性遗忘？
- [ ] LoRA rank 改大改小有什么影响？

---

## 🏆 验收标准

- [ ] 微调后模型在特定任务上明显优于基础模型
- [ ] LoRA adapter 文件 < 50MB
- [ ] 完成对比评估报告
