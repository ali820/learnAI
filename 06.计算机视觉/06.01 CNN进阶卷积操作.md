# CNN 进阶卷积操作

> 标准卷积之外还有哪些花样？1×1、深度可分离、分组卷积——这些是现代 CV 架构的基本积木。

---

## 🎯 练习目标

1. 理解 1×1 卷积为什么能"降维/升维"
2. 掌握深度可分离卷积的拆分思想
3. 算清每种卷积的参数量
4. 理解 MobileNet 为什么"轻"

## 1️⃣ 1×1 卷积：最不起眼但最有用

```python
import torch
import torch.nn as nn
import numpy as np

# 1×1 卷积 = 对每个像素做一次全连接
# 不改变空间尺寸，只改变通道数

x = torch.randn(1, 256, 14, 14)   # 256 通道的特征图

# 降维：256 → 64
conv_reduce = nn.Conv2d(256, 64, kernel_size=1)
out = conv_reduce(x)
print(f"降维: {x.shape} → {out.shape}")
print(f"参数量: {sum(p.numel() for p in conv_reduce.parameters()):,}")
# 256 × 64 × 1 × 1 + 64 = 16,448

# 对比 3×3 卷积
conv_3x3 = nn.Conv2d(256, 64, kernel_size=3, padding=1)
print(f"3×3 卷积参数量: {sum(p.numel() for p in conv_3x3.parameters()):,}")
# 256 × 64 × 3 × 3 + 64 = 147,520  ← 9 倍！
```

> 💡 ResNet 的 Bottleneck 块 = 1×1(降维) → 3×3(卷积) → 1×1(升维)。1×1 降维让 3×3 在低维空间算，大幅减少计算量。

## 2️⃣ 深度可分离卷积：MobileNet 的核心

```
标准卷积：
  输入(H,W,C_in) × 核(K,K,C_in,C_out) → 一次完成空间+通道混合
  
深度可分离卷积（拆成两步）：
  Step1 Depthwise: 每个通道单独做空间卷积    → (H,W,C_in)
  Step2 Pointwise: 1×1 卷积混合通道          → (H,W,C_out)
```

```python
# 标准卷积
conv_standard = nn.Conv2d(32, 64, 3, padding=1)

# 深度可分离卷积
depthwise = nn.Conv2d(32, 32, 3, padding=1, groups=32)  # groups=C_in
pointwise = nn.Conv2d(32, 64, 1)

# 参数对比
standard_params = sum(p.numel() for p in conv_standard.parameters())
dw_params = sum(p.numel() for p in depthwise.parameters()) + \
            sum(p.numel() for p in pointwise.parameters())

print(f"标准卷积参数: {standard_params:,}")
print(f"深度可分离:    {dw_params:,}")
print(f"压缩比: {dw_params/standard_params:.2%}")
# 输出约 11% — 几乎是十分之一！
```

## 3️⃣ 分组卷积：ResNeXt 的并行之道

```python
# 分组卷积：把通道分成 g 组，每组独立卷积
# groups=1 → 标准卷积
# groups=C_in → Depthwise 卷积

x = torch.randn(1, 64, 28, 28)

for groups in [1, 2, 4, 8]:
    conv = nn.Conv2d(64, 64, 3, padding=1, groups=groups)
    params = sum(p.numel() for p in conv.parameters())
    print(f"groups={groups:2d}: 参数量={params:,}")

print("\n💡 分组越多 → 参数越少 → 但组间信息不流通")
print("   ResNeXt: groups=32, 用 1×1 卷积在组间交换信息")
```

## 4️⃣ 🔥 三种卷积的对比总结

```python
# 手动计算参数量（不依赖 PyTorch）
def conv_params(c_in, c_out, k, groups=1):
    return c_in // groups * c_out * k * k + c_out

print("=== 输入 256ch → 输出 128ch, 空间 14×14 ===")
print(f"标准卷积 3×3:      {conv_params(256, 128, 3):,}")
print(f"标准卷积 1×1:      {conv_params(256, 128, 1):,}")
print(f"深度可分离 3×3+1×1: {conv_params(256, 256, 3, 256) + conv_params(256, 128, 1):,}")
print(f"分组卷积 g=8, 3×3:  {conv_params(256, 128, 3, 8):,}")
```

## 5️⃣ MobileNet 微型实现

```python
class MobileNetBlock(nn.Module):
    """MobileNetV1 的基础块"""
    def __init__(self, c_in, c_out, stride=1):
        super().__init__()
        self.conv = nn.Sequential(
            # Depthwise
            nn.Conv2d(c_in, c_in, 3, stride, padding=1, groups=c_in, bias=False),
            nn.BatchNorm2d(c_in),
            nn.ReLU6(inplace=True),
            # Pointwise
            nn.Conv2d(c_in, c_out, 1, bias=False),
            nn.BatchNorm2d(c_out),
            nn.ReLU6(inplace=True),
        )
    
    def forward(self, x):
        return self.conv(x)

block = MobileNetBlock(32, 64)
x = torch.randn(1, 32, 56, 56)
out = block(x)
print(f"MobileNet Block: {x.shape} → {out.shape}")
print(f"参数量: {sum(p.numel() for p in block.parameters()):,}")
```
