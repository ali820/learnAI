# 🔌 Obsidian 插件配置指南

> AI 学习专属的 Obsidian 工作流配置手册。

---

## 📦 一键安装清单

以下插件请在 Obsidian 设置 → 第三方插件 → 社区插件中搜索安装。

### 🥇  Tier 1：必装（学习刚需）

| 插件             | 搜索关键词        | 为什么需要                     | 配置要点                   |
| -------------- | ------------ | ------------------------- | ---------------------- |
| **Dataview**   | `dataview`   | 用 SQL 查询管理所有学习笔记，自动生成进度看板 | 开启 JavaScript 查询       |
| **Templater**  | `templater`  | 创建"每日学习笔记"、"概念卡片"等模板      | 设置模板文件夹为 `99.资源与工具/模板` |
| **Excalidraw** | `excalidraw` | 画神经网络架构图、流程图、思维导图         | 设置自动保存为 PNG            |
| **Kanban**     | `kanban`     | 看板管理学习任务                  | 创建 `学习看板` 面板           |

### 🥈 Tier 2：效率利器

| 插件 | 用途 | 配置要点 |
|------|------|---------|
| **Latex Suite** | 快速输入 $\frac{\partial L}{\partial w}$ 等公式 | 开启自动补全 |
| **Spaced Repetition** | 将笔记转为闪卡，间隔重复 | 在笔记中添加 `#card` 标记 |
| **Annotator** | 在 Obsidian 内标注 PDF 论文 | 配置 PDF 存储路径 |
| **Code Preview** | 预览 Python 代码输出 | — |
| **Obsidian Git** | 自动备份笔记到 GitHub | 设置每 15 分钟自动提交 |
| **Image Toolkit** | 查看图片缩放、旋转 | 默认即可 |
| **Table of Contents** | 自动生成笔记目录 | — |

### 🥉 Tier 3：提升体验

| 插件 | 用途 |
|------|------|
| **Style Settings** | 自定义主题 |
| **Calendar** | 日历视图（查看学习热力图） |
| **Iconize** | 为文件夹添加图标 |
| **Homepage** | 设置启动时打开路线图 |
| **Pomodoro Timer** | 番茄钟计时，保持专注 |

## ⚙️ 工作流配置

### 1. 模板系统 (Templater)

创建模板文件夹 `99.资源与工具/模板`，添加以下模板：

**概念卡片模板** → `概念卡片.md`
```markdown
---
created: <% tp.date.now("YYYY-MM-DD") %>
tags:
  - concept
  - <% tp.system.suggester(["ML","DL","NLP","Math"],["ML","DL","NLP","Math"]) %>
---

# <% tp.file.title %>

## 一句话解释

> 

## 核心公式

$$
$$

## 直觉理解

## 代码实现

## 相关概念

```

**每日学习笔记模板** → `每日学习.md`
```markdown
---
created: <% tp.date.now("YYYY-MM-DD") %>
tags:
  - daily
---

# <% tp.date.now("YYYY-MM-DD") %> 学习日志

## 🎯 今日目标
- [ ] 

## 📝 学习内容

## 💻 实践记录

## 💡 收获与疑问

## ⏭ 明日计划
```

### 2. Dataview 进度看板

在 `00.规划/` 中创建自动汇总：

```dataview
TABLE
  phase as "阶段",
  status as "状态",
  duration as "时长"
FROM "00.规划"
WHERE phase
SORT phase asc
```

### 3. Git 自动备份

```bash
# 在 vault 根目录初始化
git init
# 设置远程仓库
git remote add origin https://github.com/<你的用户名>/<仓库名>.git
```

然后在 Obsidian Git 插件设置中：
- 自动提交间隔：15 分钟
- 自动 Push：开启

### 4. 标签体系

建议使用统一的标签体系来组织知识：

```
#AI-learning          — 主标签
  ├── #concept        — 概念笔记
  ├── #code           — 代码笔记
  ├── #paper          — 论文笔记
  ├── #project        — 项目笔记
  └── #daily          — 每日学习日志
  
#status
  ├── #status/seed    — 初稿，需要完善
  ├── #status/growing — 正在丰富
  └── #status/evergreen — 成熟的永久笔记
```
