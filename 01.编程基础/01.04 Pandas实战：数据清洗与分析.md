## 📝 练习清单

### 1. DataFrame 基础
```python
import pandas as pd
import numpy as np

# 从字典创建
df = pd.DataFrame({
    'name': ['Alice', 'Bob', 'Charlie', 'David'],
    'age': [25, 30, 35, 28],
    'score': [85, 92, 78, 95]
})

# 查看数据
print(df.head())
print(df.info())
print(df.describe())

# 基本操作
print(df['name'])         # 取一列
print(df[['name', 'score']])  # 取多列
print(df.iloc[0:2])       # 按位置取行
print(df.loc[df['score'] > 80])  # 按条件筛选
```

### 2. 数据读取与写入
```python
# 读取 CSV（使用 Titanic 数据集）
df = pd.read_csv('titanic.csv')
# 或从 URL 读取
# url = 'https://raw.githubusercontent.com/datasciencedojo/datasets/master/titanic.csv'
# df = pd.read_csv(url)

# 写入
df.to_csv('processed_data.csv', index=False)
df.to_json('data.json', orient='records')
```

### 3. 数据清洗核心操作
```python
# 检查缺失值
print(df.isnull().sum())

# 处理缺失值
df['Age'].fillna(df['Age'].median(), inplace=True)  # 用中位数填充
df.dropna(subset=['Embarked'], inplace=True)         # 删除缺失行

# 检查并处理重复值
print(df.duplicated().sum())
df.drop_duplicates(inplace=True)

# 数据类型转换
df['Age'] = df['Age'].astype(int)
```

### 4. 分组与聚合
```python
# GroupBy 操作
# 按性别分组，计算年龄和票价的平均值
result = df.groupby('Sex')[['Age', 'Fare']].mean()
print(result)

# 多级分组
result = df.groupby(['Sex', 'Pclass'])['Survived'].mean()
print(result)

# 多个聚合函数
result = df.groupby('Pclass')['Fare'].agg(['mean', 'std', 'min', 'max'])
```

### 5. 数据透视表
```python
# 透视表：分析不同舱位和性别的生存率
pivot = pd.pivot_table(
    df,
    values='Survived',
    index='Pclass',
    columns='Sex',
    aggfunc='mean'
)
print(pivot)
```

### 6. 数据合并
```python
# concat: 纵向拼接
df1 = pd.DataFrame({'A': [1, 2], 'B': [3, 4]})
df2 = pd.DataFrame({'A': [5, 6], 'B': [7, 8]})
pd.concat([df1, df2], axis=0)

# merge: SQL-style 合并
left = pd.DataFrame({'key': ['K0', 'K1'], 'val1': [1, 2]})
right = pd.DataFrame({'key': ['K0', 'K2'], 'val2': [3, 4]})
pd.merge(left, right, on='key', how='inner')
```

## 🔥 综合实战：Titanic 数据清洗

将以上技能串联起来，完成 Titanic 数据集的完整清洗：

```python
# 1. 加载数据
df = pd.read_csv('titanic.csv')

# 2. 查看基本信息
print(df.info())

# 3. 处理缺失值
# Age: 用中位数填充
# Cabin: 缺失太多，创建 "是否有 Cabin" 特征
# Embarked: 用众数填充

# 4. 特征工程
df['FamilySize'] = df['SibSp'] + df['Parch'] + 1
df['Title'] = df['Name'].str.extract(r' ([A-Za-z]+)\.', expand=False)

# 5. 类别编码
df['Sex'] = df['Sex'].map({'male': 0, 'female': 1})
df = pd.get_dummies(df, columns=['Embarked'], drop_first=True)

# 6. 保存清洗后的数据
df.to_csv('titanic_clean.csv', index=False)

# 7. 输出清洗报告
print(f"原始数据行数: {len(df)}")
print(f"缺失值统计:\n{df.isnull().sum()}")
print(f"最终特征: {list(df.columns)}")
```
