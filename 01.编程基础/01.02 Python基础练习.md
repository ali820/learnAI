# Python 基础练习

> 目标：巩固 Python 基础语法，确保能流畅处理数据。
> 环境：在 Jupyter Notebook 或 VS Code 中完成。

## 📝 练习清单

### 1️⃣ 数据类型与控制流
```python
# 1. 创建一个列表，包含 1-10 的平方数
# 2. 用 for 循环打印出所有偶数
# 3. 用列表推导式生成 [0, 1, 4, 9, 16, ...]
# 4. 写一个函数，判断一个数是否为质数
```

### 2️⃣ 函数与 Lambda
```python
# 1. 写一个函数计算 Fibonacci 数列的第 n 项
# 2. 用 map + lambda 将列表 [1,2,3,4,5] 每个元素翻倍
# 3. 用 filter 过滤出列表中的正数
```

### 3️⃣ 文件操作
```python
# 1. 读取一个 CSV 文件（可用 Titanic 数据集）
# 2. 将处理后的数据保存为 JSON
# 3. 用 with 语句安全读写文件
```

### 4️⃣ NumPy 热身
```python
# 1. 创建 3x3 的随机矩阵
# 2. 计算两个向量的点积
# 3. 矩阵乘法：A(3x4) @ B(4x2)
# 4. 计算矩阵的转置和逆
```

### 5️⃣ Pandas 热身
```python
# 1. 创建 DataFrame 包含 5 个人的姓名、年龄、分数
# 2. 读取 CSV 文件显示前 5 行

```
