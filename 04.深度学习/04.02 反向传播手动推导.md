# 反向传播手动推导

> 这可能是 AI 学习中最"劝退"的一关。别怕，一步步手算完，你会发现它是纸老虎。

---

## 🎯 练习目标

1. 理解计算图：把网络画成一棵树
2. 手动推导一个 2 层网络的完整梯度
3. 用数值梯度验证解析梯度的正确性
4. 理解为什么反向传播比前向计算梯度快 N 倍

---

## 1️⃣ 计算图：把网络拆成原子操作

```
输入: x₁, x₂
权重: w₁, w₂
标签: y

计算图:
x₁ ─┐
    ├─→ h = w₁x₁ + w₂x₂  ──→ ŷ = sigmoid(h)  ──→ L = (ŷ - y)²
x₂ ─┘
```

```python
import numpy as np

# 前向传播
x1, x2 = 1.0, 2.0
w1, w2 = 0.5, -0.3
y = 1.0

# 一步一步算
h = w1 * x1 + w2 * x2                # 线性组合
y_hat = 1 / (1 + np.exp(-h))         # sigmoid
L = (y_hat - y) ** 2                  # MSE

print(f"h = {h:.4f}")
print(f"ŷ = {y_hat:.4f}")
print(f"L = {L:.4f}")
```

## 2️⃣ 手动推导：链式法则一层层回荡

```
       L = (ŷ - y)²
       ↓  ∂L/∂ŷ = 2(ŷ - y)
       ŷ = σ(h)
       ↓  ∂L/∂h = ∂L/∂ŷ · σ'(h) = 2(ŷ-y) · ŷ(1-ŷ)
       h = w₁x₁ + w₂x₂
       ↓  ∂L/∂w₁ = ∂L/∂h · ∂h/∂w₁ = ∂L/∂h · x₁
       ↓  ∂L/∂w₂ = ∂L/∂h · ∂h/∂w₂ = ∂L/∂h · x₂
```

```python
# 解析梯度（手工计算）
dL_dyhat = 2 * (y_hat - y)
dsigmoid = y_hat * (1 - y_hat)       # σ'(h)
dL_dh = dL_dyhat * dsigmoid
dL_dw1 = dL_dh * x1
dL_dw2 = dL_dh * x2

print("=== 解析梯度 ===")
print(f"∂L/∂w₁ = {dL_dw1:.6f}")
print(f"∂L/∂w₂ = {dL_dw2:.6f}")
```

## 3️⃣ 数值梯度验证（确保你没算错）

```python
def numerical_gradient(f, w1, w2, x1, x2, y, eps=1e-6):
    """用有限差分近似梯度：f'(x) ≈ (f(x+ε) - f(x-ε)) / 2ε"""
    L_plus = f(w1 + eps, w2, x1, x2, y)
    L_minus = f(w1 - eps, w2, x1, x2, y)
    dw1 = (L_plus - L_minus) / (2 * eps)
    
    L_plus = f(w1, w2 + eps, x1, x2, y)
    L_minus = f(w1, w2 - eps, x1, x2, y)
    dw2 = (L_plus - L_minus) / (2 * eps)
    return dw1, dw2

def loss_fn(w1, w2, x1, x2, y):
    h = w1 * x1 + w2 * x2
    y_hat = 1 / (1 + np.exp(-h))
    return (y_hat - y) ** 2

num_dw1, num_dw2 = numerical_gradient(loss_fn, w1, w2, x1, x2, y)

print("=== 数值梯度（基准） ===")
print(f"∂L/∂w₁ ≈ {num_dw1:.6f}")
print(f"∂L/∂w₂ ≈ {num_dw2:.6f}")

print("\n=== 误差验证 ===")
print(f"w₁ 误差: {abs(dL_dw1 - num_dw1):.2e}  {'✅' if abs(dL_dw1 - num_dw1) < 1e-5 else '❌'}")
print(f"w₂ 误差: {abs(dL_dw2 - num_dw2):.2e}  {'✅' if abs(dL_dw2 - num_dw2) < 1e-5 else '❌'}")

print("\n💡 数值梯度 = 真理。解析梯度 = 你推导的结果。二者一致 → 推导正确！")
```

## 4️⃣ 🔥 两层网络的完整反向传播推导

```
[X] → [Linear + ReLU] → [Linear + Softmax] → CrossEntropy Loss

Layer 2 (输出层):
  z₂ = a₁ W₂ + b₂
  ŷ  = softmax(z₂)
  L  = -Σ y log ŷ
  
  ∂L/∂z₂ = ŷ - y              ← Softmax + CE 的合体魔法
  ∂L/∂W₂ = a₁^T · (∂L/∂z₂)
  ∂L/∂b₂ = ∂L/∂z₂ 的和

Layer 1 (隐藏层):
  z₁ = X W₁ + b₁
  a₁ = ReLU(z₁)
  
  ∂L/∂a₁ = (∂L/∂z₂) · W₂^T
  ∂L/∂z₁ = ∂L/∂a₁ ⊙ ReLU'(z₁)   ← ⊙ 逐元素乘法
  ∂L/∂W₁ = X^T · (∂L/∂z₁)
  ∂L/∂b₁ = ∂L/∂z₁ 的和
```

## 5️⃣ 梯度消失 vs 梯度爆炸

```python
# 模拟：Sigmoid 堆叠 10 层后梯度还剩多少？
z = np.linspace(-5, 5, 100)
grad = z * (1 - sigmoid(z))  # sigmoid 的导数

print("Sigmoid 导数的最大值:")
print(f"  max σ'(z) = {grad.max():.4f}")
print(f"  10 层后: {0.25**10:.2e}")
print(f"  20 层后: {0.25**20:.2e}")
print("\n💡 这就是为什么 ReLU 替换了 Sigmoid 做隐藏层激活:")
print("   ReLU 导数 = 0 或 1，不会指数级衰减")
```
