# P6-用户分群

> 没有标签，如何发现数据中的隐藏结构？用聚类做用户画像。

---

## 🎯 项目目标

1. 掌握无监督学习的完整流程
2. 学会聚类结果的业务解读
3. 用 PCA 降维让聚类可视化
4. 产出可落地的用户分群报告

---

## 📦 数据

[Kaggle Mall Customers](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)

或 Sklearn 内置数据：
```python
from sklearn.datasets import make_blobs, load_wine, fetch_olivetti_faces
```

---

## 📋 任务步骤

### Step 1：数据探索
- [ ] 查看各特征的分布
- [ ] 检查是否需要标准化（聚类必须标准化！）
- [ ] 画 Pairplot 初步观察自然分组

### Step 2：K-Means 聚类
- [ ] 用肘部法则 + 轮廓系数选最佳 K
- [ ] K-Means 聚类并可视化
- [ ] 分析每个簇的特征画像

### Step 3：DBSCAN 对比
- [ ] 调 eps 和 min_samples
- [ ] 对比 K-Means 和 DBSCAN 的结果
- [ ] 解释为什么某些点被标记为噪声

### Step 4：业务解读
- [ ] 给每个簇取一个业务名称（如"高收入低消费"）
- [ ] 为每个簇推荐营销策略
- [ ] 输出用户分群报告

---

## 🏆 评估标准（无监督的特殊性）

| 维度 | 要求 |
|------|------|
| 技术 | 能解释为什么选这个 K 值 |
| 可视化 | 至少 3 张图（肘部图 + 聚类散点图 + 雷达图） |
| 业务 | 每个簇有名字和可执行的策略建议 |

---

## 📊 输出要求

1. Jupyter Notebook
2. 用户分群总结表（表格 + 雷达图）
3. 笔记：如果给你决策权，你会对哪个用户群做什么？
