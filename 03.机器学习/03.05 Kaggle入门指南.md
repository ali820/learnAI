### 3. Titanic 快速提交流程

```python
# 在 Kaggle Notebook 或本地运行
import pandas as pd
from sklearn.ensemble import RandomForestClassifier
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score

# 加载数据
train = pd.read_csv('train.csv')
test = pd.read_csv('test.csv')

# 快速特征工程
def preprocess(df):
    df = df.copy()
    df['Sex'] = df['Sex'].map({'male': 0, 'female': 1})
    df['Age'] = df['Age'].fillna(df['Age'].median())
    df['Fare'] = df['Fare'].fillna(df['Fare'].median())
    df['Embarked'] = df['Embarked'].fillna('S')
    df = pd.get_dummies(df, columns=['Embarked'], drop_first=True)
    df['FamilySize'] = df['SibSp'] + df['Parch'] + 1
    return df

train_processed = preprocess(train)
test_processed = preprocess(test)

# 选特征
features = ['Pclass', 'Sex', 'Age', 'Fare', 'FamilySize',
            'Embarked_Q', 'Embarked_S']
X = train_processed[features]
y = train['Survived']

# 训练
model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X, y)

# 预测 + 提交
test_pred = model.predict(test_processed[features])
submission = pd.DataFrame({
    'PassengerId': test['PassengerId'],
    'Survived': test_pred
})
submission.to_csv('submission.csv', index=False)
print("submission.csv 已生成！去 Kaggle 上传吧 🚀")
```

### 4. 提交
在比赛页面 → Submit Prediction → 上传 `submission.csv` → 看到你的排名！

## 📈 从入门到拿牌

### 提分路线
```
Baseline（随机森林基础特征）  ~0.75
    ↓ 特征工程更多特征
    ↓ 交叉验证选参数
    ↓ 集成多模型（Stacking）
    ↓ AutoML（AutoGluon/H2O）
Top 10%  ~0.80+
```

### 关键技能
| 技能 | 对提分的影响 |
|------|------------|
| 特征工程 | 🔴 最大 |
| 模型选择 | 🟡 中等 |
| 超参数调优 | 🟡 中等 |
| 集成学习 | 🟢 锦上添花 |
