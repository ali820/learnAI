# P18-多模态应用

> 用 CLIP + LLM 构建一个能"看懂图片"的 AI 应用。

---

## 🎯 项目目标

1. 实现图文互搜系统
2. 构建多模态问答 Demo
3. 部署上线

---

## 📋 任务步骤

### Step 1：图文搜索
- [ ] 用 CLIP 对图片库生成 Embedding
- [ ] 实现"以文搜图"：输入文字 → 找到最匹配的图片
- [ ] 实现"以图搜图"：上传图片 → 找相似图片

### Step 2：多模态 QA
- [ ] 接入 LLaVA / GPT-4V API
- [ ] 上传图片 + 提问 → 返回答案

### Step 3：UI + 部署
- [ ] Gradio/Streamlit UI
- [ ] Docker 部署

---

## 🏆 验收标准

- [ ] 以文搜图准确率 > 80%（Top-5）
- [ ] 多模态 QA 能正确回答关于图片的问题
