## 1️⃣ 向量与几何直觉

向量不是抽象符号，它代表空间中的一个点（或方向）。

```python
import numpy as np
import matplotlib.pyplot as plt

# 两个向量
a = np.array([3, 2])
b = np.array([1, 3])

# 可视化
fig, ax = plt.subplots(figsize=(6, 6))
ax.quiver(0, 0, a[0], a[1], angles='xy', scale_units='xy', scale=1, color='red', label='a')
ax.quiver(0, 0, b[0], b[1], angles='xy', scale_units='xy', scale=1, color='blue', label='b')

# 向量加法
c = a + b  # [4, 5]
ax.quiver(0, 0, c[0], c[1], angles='xy', scale_units='xy', scale=1, color='green', label='a+b')

ax.set_xlim(-1, 6)
ax.set_ylim(-1, 6)
ax.axhline(y=0, color='k', linewidth=0.5)
ax.axvline(x=0, color='k', linewidth=0.5)
ax.set_aspect('equal')
ax.grid(True, alpha=0.3)
ax.legend()
plt.show()

# 点积 = a·b = |a||b|cos(θ)
dot = np.dot(a, b)        # sum(a[i] * b[i]) = 3*1 + 2*3 = 9
cos_theta = dot / (np.linalg.norm(a) * np.linalg.norm(b))
theta = np.arccos(cos_theta)
print(f"a·b = {dot}, 夹角 = {np.degrees(theta):.1f}°")
```

> 💡 **关键直觉**：点积越大 → 两向量方向越接近。在深度学习中，这决定了两个样本、两个词向量的相似度！

## 2️⃣ 矩阵乘法 = 线性变换

矩阵乘法 `W @ x` 把一个向量转换到新空间 —— **这就是神经网络每一层做的事**。

```python
# 旋转矩阵：将向量逆时针旋转 θ 弧度
theta = np.pi / 4  # 45 度
R = np.array([
    [np.cos(theta), -np.sin(theta)],
    [np.sin(theta), np.cos(theta)]
])

x = np.array([1, 0])   # 沿 x 轴的单位向量
y = R @ x              # 旋转 45 度 → [0.707, 0.707]

# 对一个正方形做旋转
square = np.array([
    [0, 0],
    [1, 0],
    [1, 1],
    [0, 1],
    [0, 0]   # 闭合
])
rotated = square @ R.T  # 每行是一个点

# 可视化（练习时自己画出来！）
```

> 💡 **关键直觉（Phase 3 预习）**：神经网络的每一层就是 `激活函数(W @ x + b)`。`W @ x` 是线性变换（旋转+缩放），`+b` 是平移，`激活函数` 把直线变弯。

## 3️⃣ 特征值与特征向量

```python
# 特征值分解：A v = λ v
A = np.array([[4, 2],
              [1, 3]])

eigenvalues, eigenvectors = np.linalg.eig(A)

print("特征值 λ：", eigenvalues)
print("特征向量 v (每列一个)：")
print(eigenvectors)

# 验证：A @ v = λ v
for i in range(2):
    v = eigenvectors[:, i]
    lam = eigenvalues[i]
    left = A @ v
    right = lam * v
    print(f"\n特征向量 {i}: A@v = {left}, λv = {right}")
    print(f"误差: {np.linalg.norm(left - right):.2e}")
```

> 💡 **直觉**：特征向量是矩阵 A 作用后**方向不变**的特殊向量（只会被拉伸/压缩 λ 倍）。PCA 就是在找数据协方差矩阵的"主特征向量"。

## 4️⃣ SVD 分解（选做）

```python
# SVD: A = U Σ V^T
A = np.random.randn(5, 3)
U, S, Vt = np.linalg.svd(A, full_matrices=False)

# 验证
reconstructed = U @ np.diag(S) @ Vt
print(f"重构误差: {np.linalg.norm(A - reconstructed):.2e}")
```

> 💡 奇异值 S 表示数据在每个"方向"上的重要程度。SVD 是推荐系统、图像压缩的基础。
