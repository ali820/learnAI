# NumPy 实战：矩阵运算与线性代数

## 练习目标

1. 熟练创建和操作多维数组
2. 掌握矩阵运算（点积、转置、求逆）
3. 理解广播机制
4. 用 NumPy 实现基础线性代数运算

## 📝 练习清单

### 1. 数组创建与属性
```python
import numpy as np

# 创建不同类型的数组
a = np.array([1, 2, 3])
b = np.zeros((3, 4))
c = np.ones((2, 3))
d = np.eye(4)           # 单位矩阵
e = np.arange(10)       # [0, 1, ..., 9]
f = np.linspace(0, 1, 5)  # [0, 0.25, 0.5, 0.75, 1.0]
g = np.random.randn(3, 3) # 正态分布随机数

# 属性
print(a.shape, a.ndim, a.dtype)
```

### 2. 索引与切片
```python
# 基础索引
mat = np.arange(12).reshape(3, 4)
print(mat[0, :])    # 第一行
print(mat[:, 1])    # 第二列
print(mat[1:3, :])  # 第二到第三行

# 布尔索引
print(mat[mat > 5])

# Fancy 索引
print(mat[[0, 2], :])
```

### 3. 矩阵运算
```python
A = np.random.randn(3, 4)
B = np.random.randn(4, 2)

# 矩阵乘法
C = A @ B          # Python 3.5+ 语法
C = np.dot(A, B)   # 等效写法

# 逐元素运算
print(A * 2)       # 标量乘法
print(A + A)       # 矩阵加法

# 转置与逆
X = np.random.randn(3, 3)
print(X.T)                        # 转置
print(np.linalg.inv(X))           # 逆矩阵
print(X @ np.linalg.inv(X))       # 应为近似单位矩阵
```

### 4. 广播机制
```python
# 形状不同的数组自动扩展
a = np.array([1, 2, 3])       # shape (3,)
b = np.array([[1], [2], [3]]) # shape (3, 1)
print(a + b)                   # 广播后 shape (3, 3)

# 标准化示例
data = np.random.randn(5, 4)
mean = data.mean(axis=0)
std = data.std(axis=0)
normalized = (data - mean) / std  # 广播的应用
```

### 5. 线性代数实战：解方程组
```python
# 解线性方程组 Ax = b
A = np.array([[2, 1], [1, 3]])
b = np.array([5, 6])

x = np.linalg.solve(A, b)
print(x)  # 应得到 [1.8, 1.4]
```

### 6. 从零实现线性回归
```python
# 用 NumPy 实现 y = Xw + b 的线性回归
# 普通最小二乘法：w = (X^T X)^{-1} X^T y
np.random.seed(42)
X = np.random.randn(100, 1)
true_w, true_b = 2.5, 1.3
y = true_w * X + true_b + 0.1 * np.random.randn(100, 1)

# 添加偏置项
X_b = np.c_[np.ones((100, 1)), X]

# 闭式解
w = np.linalg.inv(X_b.T @ X_b) @ X_b.T @ y
print(f"w = {w[1][0]:.3f}, b = {w[0][0]:.3f}")  # 应接近 2.5, 1.3
```
