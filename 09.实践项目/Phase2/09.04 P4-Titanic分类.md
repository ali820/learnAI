# P4-Titanic 分类

> 机器学习界的"Hello World"。用 Sklearn 完整流程预测泰坦尼克号乘客生存。

---

## 🎯 项目目标

1. 走通一个完整的 ML 分类 pipeline
2. 掌握特征工程的核心技巧
3. 学会模型对比与调优
4. 提交 Kaggle 并看到排名

---

## 📦 数据

[Kaggle Titanic](https://www.kaggle.com/c/titanic/data) → 下载 `train.csv` 和 `test.csv`

## 📋 任务步骤

### Step 1：探索性分析（EDA）
- [ ] 查看数据形状、缺失值、数据类型
- [ ] 分析各特征与 Survived 的关系
- [ ] 画至少 3 张有洞察的可视化图

### Step 2：特征工程
```python
# 建议至少完成以下特征：
- [ ] 从 Name 提取 Title（Mr/Mrs/Miss/...）
- [ ] Age 分段（儿童/青年/中年/老年）
- [ ] FamilySize = SibSp + Parch + 1
- [ ] IsAlone = (FamilySize == 1)
- [ ] Fare 分箱
- [ ] Cabin 处理（有 Cabin vs 无 Cabin）
```
