# P19-MLOps 流水线

> 搭建一条完整的 ML 流水线：实验→训练→部署→监控。

---

## 🎯 项目目标

1. 用 MLflow 管理实验
2. 用 Docker + FastAPI 部署模型
3. 实现 CI/CD（GitHub Actions）
4. 搭建监控面板

---

## 📋 任务步骤

### Step 1：实验管理
- [ ] 用 MLflow 追踪至少 10 组实验
- [ ] 对比不同模型的性能和参数
- [ ] 注册最佳模型到 Model Registry

### Step 2：自动化部署
- [ ] 写 FastAPI 服务
- [ ] Docker 化
- [ ] GitHub Actions → push 自动构建 + 测试

### Step 3：监控
- [ ] 用 Prometheus 收集指标
- [ ] 用 Grafana 可视化
- [ ] 设置告警（模型准确率下降 → 通知）

---

## 🏆 验收标准

- [ ] 端到端流水线可一键运行
- [ ] 新模型自动部署
- [ ] 监控面板显示实时指标
